This is a Web Application developed by Shreyas Salankimatt.
It uses python and jQuery AJAX (Async Javascript and XML) to generate PDF reports for individual students based on a given excel sheet.
It Utilizes SMTPLib (python) to send emails and Flask (python) for web and backend processing and ReportLab (python) to generate PDFs.
